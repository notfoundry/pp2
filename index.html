Defines

PP2_EAT(...)#

Substitutes none of the provided macro arguments into the macro replacment list, causing all tokens in those arguments to be erased.

Parameters

  • ... – the token sequence to erase

PP2_SCAN(...)#

Substitutes all of the provided macro arguments into the macro replacement list without changing them.

This is typically used to force an additional scan over the tokens in the macro arguments, allowing additional macro expansions to take place if any are legal to perform. This only differs from PP2_OPEN in its intended use, as demonstrated: 

#define FOO(x) PP2_SCAN(x)
#define BAR 123
FOO(BAR PP2_BLOCK ()) // 123

// compared to...

#define FOO(x) PP2_OPEN x
FOO((1,2,3)) // 1,2,3

Parameters

  • ... – the token sequence to scan

PP2_OPEN(...)#

Removes the parentheses surrounding the provided macro arguments, and substitutes the tokens contained between those parentheses into the macro replacement list without changing them.

See PP2_SCAN for a comparison between these two macros.

Parameters

  • ... – the token sequence to have the outermost parentheses stripped from

PP2_BLOCK#

Substitutes no tokens into the macro replacmeent list, effectively advancing the macro evaluator ahead by one token and performing a no-op.
