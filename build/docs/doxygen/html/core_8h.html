◆ PP2_SCAN

#define PP2_SCAN (   ...)    __VA_ARGS__

Substitutes all of the provided macro arguments into the macro replacement list without changing them.

This is typically used to force an additional scan over the tokens in the macro arguments, allowing additional macro expansions to take place if any are legal to perform. This only differs from PP2_OPEN in its intended use, as demonstrated:

#define FOO(x) PP2_SCAN(x)
#define BAR 123
FOO(BAR PP2_BLOCK ()) // 123
// compared to...
#define FOO(x) PP2_OPEN x
FOO((1,2,3)) // 1,2,3
